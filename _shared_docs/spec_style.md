# Robin Identity OS — spec_style_v0

## Philosophy

This is not a website.
This is a system interface with taste.

Principles:

- calm, deep, selective
- no visual noise
- no unnecessary decoration
- restraint over expression
- subtraction over addition

---

## Typography System

### Font Roles

1. Display (Hero only)

- Serif
- Elegant, high contrast
- Used ONLY for:
  - "Engineer. Executive. Mother."
  - "Flow. Capital. Token."

1. Section Titles

- SAME serif as display
- Smaller than hero
- Consistent across all sections

1. Labels (eyebrow text)

- Sans-serif
- Uppercase
- Letter-spacing increased
- Small size

Examples:

- MS. ROBIN XIE
- BOOKS
- SIGNAL
- CURRENT LENS

1. Body Text

- Sans-serif
- Clean and readable
- Same size across entire site

1. Buttons

- Same font as body
- Slightly tighter spacing

---

## Typography Rules

- NEVER mix serif and sans inside the same line
- NEVER use different serif fonts across sections
- "Engineer. Executive. Mother." and "Flow. Capital. Token." must:
  - use SAME font
  - use SAME size
  - differ ONLY in color
- Hero lines should stay on ONE line when possible
- Avoid breaking them into stacked lines unless screen is small

---

## Size Hierarchy (relative)

- Hero Display: 100%
- Section Title: 60–70%
- Body: 35–40%
- Label: 20–25%

---

## Color System

### Base

- Deep ocean gradient background

### Text

- Soft white (#EAF6FF)

### Accent (limit to 2)

- Cyan (#00E5FF)
- Blue (#1E90FF)

---

## Layout Rules

- Max 2 columns in hero
- Max 2 primary buttons
- Remove unnecessary boxes

---

## Spacing

- Prefer more whitespace than expected
- Vertical rhythm > density

---

## Motion

- subtle only
- slow transitions
- no gimmicks

---

## Content Rules

- no “NOT X BUT Y” phrasing
- affirmative language
- concise
- no repetition across sections
